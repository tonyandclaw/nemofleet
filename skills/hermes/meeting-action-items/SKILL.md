### Usage Tips

1. Replace `YYYY-MM-DD` with actual meeting date
2. Fill in attendees list
3. For each action item:
   - **負責人**: Who is accountable (use @mention if applicable)
   - **行動項**: Clear, actionable description
   - **期限**: Specific date or timeframe (e.g., "2024-06-10" or "本週五")
4. Add rows as needed — copy the `| | | |` line
5. Use Notes section for context that doesn't fit the table
